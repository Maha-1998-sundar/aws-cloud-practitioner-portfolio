Personal Portfolio Website Deployment (AWS S3 + CloudFront)

This project demonstrates the deployment of a static personal portfolio website using Amazon S3 and Amazon CloudFront. The solution is designed to be secure, scalable, and cost-efficient.

Architecture Overview

  • Amazon S3 – Private bucket for static website files
  • Amazon CloudFront – Content Delivery Network (CDN)
  • Origin Access Control (OAC) – Secure access between CloudFront and S3
  • HTTPS enforced for secure communication

Key Features

  • Private S3 bucket (no public access)
  • Secure delivery using CloudFront
  • Low latency and high availability
  • Industry best practices followed

Purpose of This Project

This project was created as part of my AWS Cloud learning journey to gain hands-on experience with real-world cloud infrastructure and security concepts.
